# Didascalos RAG Grammar Generator

I built this Retrieval-Augmented Generation (RAG) pipeline for producing structured Ancient Greek grammar modules as part of the Didaskalos project, a frequency-based textbook of Ancient Greek. The current output needs a lot of manual revisions but it could serve as a starting point for localizable modules that can be translated to other languages such as Persian.

The RAG bot generates instructional units covering the morphological categories in a sample Didaskalos frequency-based syllabus by combining large language models with curated reference materials, primarily Smyth’s Greek Grammar and Crosby and Schaeffer’s An Introduction to Greek (forked from https://github.com/gregorycrane/CrosbySchaeffer2.0).

## Overview

The pipeline takes syllabus entries (e.g. morphological or grammatical categories) from a sample syllabus generated by Didaskalos and produces self-contained instructional modules. Each module is designed to resemble a textbook section, including explanation, paradigms, and examples.
The project requires the following Python packages:

llama-index-core  
llama-index-llms-groq  
llama-index-readers-file  
llama-index-embeddings-huggingface  
llama-index-embeddings-instructor

## External Resources

This project depends on local copies of the following works:

- Smyth, Greek Grammar
- Crosby and Schaeffer, An Introduction to Greek

These must be forked or cloned from the relevant GitHub repositories and made available locally for ingestion into the vector index. The pipeline will not function without these source texts!
You should also create a .env file with your Groq API key:
GROQ_API_KEY=your_api_key_here

Batch processing supports rate limiting, timeouts, and progress tracking.
The output is in markdown.
